System Information
Welcome to the Fire Department Blood Pressure Management System!

This Django-based system provides comprehensive blood pressure monitoring and management for firefighters.

Available Features:
  • REST API: Complete API endpoints at /api/
  • Patient Management: Full CRUD operations for firefighter patients
  • BP Readings: Automated categorization and abnormal reading detection
  • Workflow Tasks: Hypertension management workflow automation
  • Communication Logs: Advanced filtering and analytics
  • Admin Interface: Django admin panel for data management
Getting Started:
  1. Run migrations: python manage.py migrate
  2. Load sample data: python manage.py seed_data
  3. Access API at: /api/
  4. Admin panel: /admin/
API Authentication:

Use session authentication or login via POST /api/auth/login/
